marv → WebAssembly: capability-gated sandbox

Each module below was compiled from marv to a core WebAssembly module by marv build --target wasm-core. A capability is a host import: the page decides which to provide. A module's import list is its capability manifest — statically inspectable, right here, before any code runs.
